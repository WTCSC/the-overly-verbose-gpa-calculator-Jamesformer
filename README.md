# The overly verbose GPA calculator
This program can generate for a six class schedule and can calculate a gpa for the first half or the last half of your classes. 

## Requirements
This project requires:
**Python version 3.12 or later**
**Git version 2.51 or later**

## Installation
You must copy the following into your terminal to install this program ; git clone (your repository url)

## Examples
Here is what the code will look like when run:

```
welcome to the rotaluclac 6 class GPA calculator
definitely getting your grades from 'jeffcopublicshools.us/infinatecampus' 
your 1st grade is 0.0
your 2nd grade  is 3.1
your 3rd grade is 3.4
your 4th grade is 0.1
your 5th grade is 0.0
your 6th grade is 3.9
based on the calculations your gpa should be around a 1.75
would you like to focus on the first half or second half of your classes(enter 1 or 2): 1
according to the calculations you should have 2.17
congratulations it seems that the gpa for the first half of your classes are higher than your overall gpa
```
